# ⚡ Быстрый деплой за 2 минуты

## Самый простой способ - Netlify:

1. **Откройте**: https://www.netlify.com
2. **Нажмите "Sign up"** (можно через Google/GitHub)
3. **На главной странице перетащите папку `blackbird`** в область "Want to deploy a new site without connecting to Git? Drag and drop your site output folder here"
4. **Готово!** Ваш сайт будет доступен по адресу типа: `amazing-site-12345.netlify.app`

### Изменить адрес сайта:
- Site settings → Domain management → Options → Edit site name
- Можно задать: `blackbird-macros.netlify.app` (или любое свободное имя)

---

## Альтернатива - Vercel (тоже очень просто):

1. **Откройте**: https://vercel.com
2. **Зарегистрируйтесь** (через GitHub проще всего)
3. **"Add New Project"** → **"Browse"** → выберите папку `blackbird`
4. **"Deploy"**
5. **Готово!** Адрес будет: `blackbird-xxx.vercel.app`
